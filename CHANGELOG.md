# 배전공사 예산 관리 시스템 변경이력

## 2026-06-05

### 손익 탭 구조 개편
- 손익 탭을 자본 탭과 동일한 구조로 전면 재작성
- 배정예산(A)이 맨 앞에 표시되도록 변경 (기존: 설계금액)
- 손익 예산현황 테이블: 자금운용사업코드, 사업명, 배정예산(A), 소비금액(B), 약정금액(C), 집행실적(D=B+C), 잔액(E=A-D), 집행율(D/A), 진행중공사비(F), 최종예상(G=D+F), 예상잔액(A-G), 예상집행율(G/A), 건수
- 배정예산(A) 직접 입력 기능 추가 (자본과 동일)
- 입력 시 잔액/집행율/예상잔액/예상집행율 자동 재계산
- 요약 카드 6개: 배정예산, 집행실적, 잔액, 진행중공사비, 최종예상, 공사건수

### Backend 추가
- `BUDGET_REVENUE` 딕셔너리: 수선비 시트 기반 16개 손익 예산항목 (사업코드, 배정예산)
- `REVENUE_CAT_MAP`: 공사관리대장 손익예산과목 → 수선비 예산항목 매핑
- `_build_comparison()` 공통 함수로 자본/손익 budget_comparison 생성 통합
- `_totals()` 공통 함수로 요약 합계 계산 통합
- 손익예산과목 파싱 시 접미사 제거 ("배전경상-경남/통영/전력공급팀" → "배전경상")
